StColoration

These images show the results of using the RenderMan Shading Language (RSL) to write a variety of special effects surface shaders. The notes and RSL code accompanying each image explain how each effect was achieved.

MatrixComposition

This page demonstrates the basics of scripting shapes RenderMan scene description (rib) protocol. Writing rib files by hand focuses attention on the use of color, opacity and the fundamental concepts related to xyz coordinate system.

 

PatternAnim

The animation and notes on this page explain how custom shaders written in the RenderMan Shading Language can be animated using SLIM parameter expressions.

LevelOfDetail

This page demonstrates the support offered by RenderMan for LOD. Three models of a short length of Spanish Moss (Tillandsia Usneoides) were made in Maya. Using LOD the renderer selects the appropriate model to render based on the area of screen space occupied by the model.

Fire

The purpose of this project was to write a surface shader that when used in conjunction with the previously written (water) displacement shader. The emphasise was on the vital need to carefully observe and document a natural phenomena before creating a digital version of it.
